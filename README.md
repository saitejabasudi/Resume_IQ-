About Resume IQ

<img width="1920" height="1056" alt="Image" src="https://github.com/user-attachments/assets/15673b3b-00be-4048-a28f-973a998dc55b" />

Resume IQ is an AI-powered resume analysis web application designed to help job seekers improve their resumes and increase their chances of getting shortlisted.

The app analyzes resume content, compares it with a job description, and provides smart insights to optimize hiring success.


---

What Users Can Do

• Upload a resume (PDF / DOC / DOCX)
• Paste a job description
• Analyze resume content instantly
• Get an ATS compatibility score
• View job match percentage
• See matched skills
• Identify missing skills


---

How the App Works

1. Extracts text from the uploaded resume


2. Processes and analyzes resume structure


3. Compares resume content with the job description


4. Calculates:

ATS Resume Score

Job Match Score



5. Highlights:

Matching skills

Missing skills





---

Core Features

• Resume file upload system
• Job description comparison engine
• Dynamic ATS score (circular visual UI)
• Job match percentage indicator
• Matched & missing skills breakdown
• Smooth loading animation
• Clean, mobile-friendly interface
• Installable Progressive Web App (PWA)

Technology Stack Used

Frontend

• React
• Tailwind CSS
• Axios

Backend

• Serverless API
• pdf-parse (PDF text extraction)
• mammoth (DOCX text extraction)
• formidable (file handling)
